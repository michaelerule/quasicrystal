To reproduce these diffraction patterns, pass a quasicrystal rendering rendered using the sum-of-plane-waves approach, apply some nonlinearities, (e.g. contrast / threshold), and then take the Fourier transform. The program ImageJ can be used for generating Fourier transforms of images.
